| Area | PySpider | GoSpider | RustSpider | JavaSpider |
| --- | --- | --- | --- | --- |
| Language | Python | Go | Rust | Java |
| Packaging | editable package | compiled binary | release binary | Maven package |

## Notes

- `PySpider` 更适合快速迭代、项目脚手架和 AI 编排。
- `GoSpider` 更适合二进制交付、并发执行和控制面集成。
- `RustSpider` 更适合强类型、高性能和 feature-gated 部署。
- `JavaSpider` 更适合 Maven 打包、浏览器工作流和企业生态集成。
