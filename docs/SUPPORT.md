# Support

Public support surfaces for this repository currently come from repository-owned documentation and issue workflows.

## Current Support Lanes

- `CONTRIBUTING.md`
- `docs/ADOPTERS.md`
- `docs/COOKBOOK.md`
- `docs/STARTERS.md`
- `docs/MARKETPLACE.md`

## Intended Use

- report runtime regressions
- share adopter and validation stories
- publish migration notes
- document extension compatibility and plugin lifecycle expectations

## Boundaries

- this repository does not claim a staffed commercial support desk
- third-party adopter evidence must remain explicit and public before stronger support claims are made
