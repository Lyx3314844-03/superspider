# SuperSpider 🕷️

四合一爬虫框架集合 - 支持 Windows、Linux、macOS 三种操作系统

## 🌟 特性

- **PySpider**: Python 高性能异步爬虫
- **RustSpider**: Rust 系统级爬虫，极致性能
- **GoSpider**: Go 并发爬虫，简洁高效
- **JavaSpider**: Java 企业级爬虫，稳定可靠

## 🚀 快速开始

### Python
```bash
cd pyspider
pip install -e .
```

### Rust
```bash
cd rustspider
cargo build
```

### Go
```bash
cd gospider
go build
```

### Java
```bash
cd javaspider
mvn install
```
