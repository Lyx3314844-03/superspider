# PySpider

PySpider 是 SuperSpider 中的 Python 爬虫框架。

## Capabilities

- `python -m pyspider` CLI
- scrapy 风格项目运行
- AI 提取与研究流程
- 浏览器和 HTTP 混合抓取
- 反爬、验证码、节点逆向、媒体与数据集输出

## Install

Windows:

- `..\scripts\windows\install-pyspider.bat`

Linux:

- `../scripts/linux/install-pyspider.sh`

macOS:

- `../scripts/macos/install-pyspider.sh`

## Run

```bash
python -m pip install -r requirements.txt
python -m pip install -e .
python -m pyspider version
```
